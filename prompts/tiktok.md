# TikTok Prompt

Transform the master thesis into short-form video assets.

## Output types
- 30-45 second script
- alternate hook bank
- on-screen text
- CTA

## Rules
- first 3 seconds must create tension
- write for spoken delivery
- keep rhythm tight
- use contrast, surprise, specificity, or consequence
- end with a direct action or thought trigger

## Deliver
- hook options
- full script
- on-screen text
- CTA
