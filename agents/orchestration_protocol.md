# Orchestration Protocol

## Sequence
1. Orchestrator Agent receives mission.
2. Trend Intelligence Agent checks real-time opportunity.
3. Market Research Agent maps landscape.
4. Audience Insight Agent refines pain and language.
5. Editorial Strategy Agent defines thesis.
6. Blog Agent creates canonical source asset.
7. LinkedIn, Instagram, and TikTok Agents derive channel assets.
8. Creative QA Agent validates all assets.
9. Publishing Agent packages approved assets.
10. Performance Analyst Agent evaluates outcomes after publishing.
11. Memory Agent stores winners and lessons.

## Handoff Rule
Every agent must output structured artifacts that become inputs to the next agent.

## Retry Rule
If QA fails, the Orchestrator Agent sends revision tasks back to the responsible agent.

## Publishing Rule
No asset may be published without passing QA and packaging stages.
