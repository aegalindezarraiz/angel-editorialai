# Multi-Agent Stack

This directory defines the specialized agentic stack for digital marketing, social media content creation, and real-time trend intelligence.

## Core Agents

1. Orchestrator Agent
2. Trend Intelligence Agent
3. Market Research Agent
4. Audience Insight Agent
5. Editorial Strategy Agent
6. LinkedIn Agent
7. Instagram Agent
8. TikTok Agent
9. Blog Agent
10. Creative QA Agent
11. Publishing Agent
12. Performance Analyst Agent
13. Memory Agent

## Operating Principle

The Orchestrator Agent receives a content mission and delegates tasks to specialized agents. Each agent returns structured outputs that become inputs for the next stage.

## Output Goal

Generate a complete multi-channel content packet that is research-backed, platform-native, trend-aware, and ready for publication.
