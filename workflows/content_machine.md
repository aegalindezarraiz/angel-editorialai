# Content Machine Workflow

## Input
A raw topic, question, case, story, client objection, legal update, market insight, or offer.

## Stage 1: Research Snapshot
Generate:
- audience problem
- desired outcome
- hidden cost of inaction
- common myth
- proof source
- content angle

## Stage 2: Strategic Thesis
Generate:
- one-sentence thesis
- 3 supporting arguments
- one contrarian twist
- one CTA path

## Stage 3: Source Asset
Create one master blog/article with:
- headline
- opening hook
- problem section
- framework section
- example section
- CTA section

## Stage 4: Channel Derivatives

### LinkedIn
- version A: authority post
- version B: story-driven post
- version C: carousel outline

### Instagram
- carousel (8 slides)
- caption
- reel concept
- visual direction

### TikTok
- 30-45 second script
- 3 second hook
- on-screen text
- CTA

### Blog
- SEO title
- slug
- meta description
- article body

## Stage 5: QA
Reject content if it is:
- generic
- repetitive
- ungrounded
- off-brand
- weak in CTA

## Stage 6: Packet Output
Deliver a content packet containing all finalized channel assets.
