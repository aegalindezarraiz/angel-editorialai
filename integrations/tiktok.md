# TikTok Integration

## Purpose
Prepare and connect short-form video publishing and reporting workflows.

## Main Operations
- upload video asset
- create draft caption package
- schedule publication
- fetch video metrics
- fetch engagement summary

## Inputs
- video asset
- caption
- hashtags
- schedule data

## Outputs
- video id
- publish status
- metrics snapshot

## Storage
- credentials
- asset registry
- metrics history
