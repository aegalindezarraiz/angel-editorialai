# Credential Matrix

## LinkedIn
- client_id
- client_secret
- redirect_uri
- access_token
- organization_id or member_urn

## Instagram
- app_id
- app_secret
- redirect_uri
- access_token
- instagram_business_account_id
- facebook_page_id

## TikTok
- client_key
- client_secret
- redirect_uri
- access_token
- advertiser or account identifier if applicable

## Blog and Website
- cms endpoint or admin url
- api token
- site id or publication id if needed

## Web Research
- search api key if external provider is used
- trend data key if external provider is used

## Analytics
- platform analytics token
- web analytics token
- property id or workspace id
