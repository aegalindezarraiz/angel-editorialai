# Blog and Web Integration

## Purpose
Publish long-form content to the website or CMS and ingest web content for research.

## Main Operations
- create article draft
- publish article
- update article
- upload media asset
- fetch page metrics
- ingest url content

## Inputs
- article title
- slug
- meta description
- body content
- media assets

## Outputs
- article id
- publication status
- page metrics snapshot
- extracted source content

## Storage
- credentials
- article registry
- source registry
- metrics history
