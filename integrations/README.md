# Integrations Layer

This directory defines the concrete resources and tools needed to connect the content machine with social networks and the web.

## Target Domains
- LinkedIn
- Instagram
- TikTok
- Blog and website
- Web research sources
- Search and trends
- Analytics and performance

## Integration Model
Each integration should define:
- purpose
- required credentials
- main operations
- expected inputs
- expected outputs
- storage requirements

## Core Rule
The content machine should separate content generation from channel transport and publishing.
