# Analytics Integration

## Purpose
Collect performance metrics from social channels and website analytics.

## Main Operations
- fetch post metrics
- fetch account metrics
- fetch page metrics
- compare content units
- identify top hooks and top formats

## Inputs
- platform identifier
- post id
- date range
- account id

## Outputs
- metrics snapshot
- comparative report
- winners and losers summary

## Storage
- metrics warehouse
- benchmark history
- performance memory feed
