# LinkedIn Integration

## Purpose
Publish and manage LinkedIn content assets.

## Main Operations
- create draft post
- publish post
- upload asset
- schedule post
- fetch post metrics
- fetch comment and engagement summary

## Inputs
- text copy
- optional media asset
- CTA
- schedule data

## Outputs
- post id
- publish status
- metrics snapshot

## Storage
- credentials
- post registry
- metrics history
