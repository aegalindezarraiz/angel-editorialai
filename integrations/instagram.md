# Instagram Integration

## Purpose
Publish and manage Instagram content assets including feed posts, carousels, and reel-related packaging.

## Main Operations
- create media container
- publish media
- upload carousel assets
- schedule publication
- fetch metrics
- fetch account level insights

## Inputs
- caption
- image or video assets
- carousel sequence
- schedule data

## Outputs
- media id
- publish status
- metrics snapshot

## Storage
- credentials
- asset registry
- metrics history
