# Skills Stack

This directory defines the operational skills stack for the content machine.

## Core Skill Layers

1. Research Skill
2. Editorial Strategy Skill
3. Long-Form Writing Skill
4. LinkedIn Skill
5. Instagram Skill
6. TikTok Skill
7. Blog Skill
8. Brand QA Skill
9. Publishing Skill
10. Analytics Memory Skill

## Execution Logic

The machine should not generate content directly from a raw topic without passing through:
- research
- thesis creation
- source asset creation
- channel transformation
- quality control
- publishing preparation
- performance memory

## Goal

Turn a single input into a coordinated, reusable, and measurable content system.
